\par

\basemodule[.main] also has optional runtime dependencies of two types,
for each type two implementations are provided and at least one of each type should be present for our program to be useful.

The first Type is IAlgoritm which is implemented by \basemodule[.algorithm.greedy] and by \basemodule[.test.algorithm.force].

The second Type is IGraphLoader which is implemented by  \basemodule[.file.json] and by \basemodule[.file.toml].
